# Support

- **Usage questions / ideas** → [GitHub Discussions](https://github.com/rnwolfe/knit/discussions).
- **Bugs / feature requests** → [open an issue](https://github.com/rnwolfe/knit/issues/new/choose) (use the forms).
- **Security issues** → see [SECURITY.md](SECURITY.md) (private reporting — do not open a public issue).
- **Diagnostics first**: run `knit doctor --for-agent --json` and include the output (token auto-redacted).

This is a community project maintained on a best-effort basis. No SLA except for security reports.
